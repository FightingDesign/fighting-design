## 🐳 Introduce

Special `svg` icon library for `fighting-design`.

## ⚒️ Install

```shell
# pnpm
pnpm add --save-dev @fighting-design/fighting-icon

# npm
npm i --save-dev @fighting-design/fighting-icon

# yarn
yarn --save-dev add @fighting-design/fighting-icon
```
