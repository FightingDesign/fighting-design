# 安装

<div>

## pnpm

使用快速的，节省磁盘空间的包管理工具 [pnpm](https://pnpm.io/zh/) 进行安装

```shell
pnpm add fighting-design
```

</div>

## npm

使用 `npm` 进行安装

```shell
npm install fighting-design
```

全局安装

```shell
npm install fighting-design --save
```

## yarn

或者使用 `yarn` 进行安装

```shell
yarn add fighting-design
```
