## Attributes

| 参数          | 说明                    | 类型          | 可选值 | 默认值 |
| ------------- | ----------------------- | ------------- | ------ | ------ |
| `list`        | 数据                    | array         | ——     | array  |
| `type`        | 布局方式                | (flex,column) | ——     | flex   |
| `colGap`      | 列间隙                  | number        | ——     | 0      |
| `rowGap`      | 行间隙                  | number        | ——     | 0      |
| `minWidth`    | 单行最小宽度            | number        | ——     | 0      |
| `cols`        | 列数,不指定最小宽度生效 | number        | ——     | 2      |
| `scrollAwait` | 滚动节流时间            | number        | ——     | 100    |

## Slots

| 名称      | 说明             |
| --------- | ---------------- |
| `default` | 瀑布流单项的内容 |

## Events

| 事件名称     | 说明       | 参数 |
| ------------ | ---------- | ---- |
| `scroll-end` | 滚动到底部 |      |
