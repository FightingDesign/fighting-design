Close Btn 关闭按钮 ​

想不到关闭按钮也可以如此精致吧？

  • 源代码
  • 文档编辑

基本使用 ​

插槽插入一个 svg-icon 即可

html
<template>
  <f-close-btn :icon="FIconCross" />
</template>

<script lang="ts" setup>
  import { FIconCross } from '@fighting-design/fighting-icon'
</script>
开启

禁用状态 ​

disabled 可禁用一个关闭按钮

html
<script lang="ts" setup>
  import { FIconCross } from '@fighting-design/fighting-icon'
</script>

<template>
  <f-close-btn disabled :icon="FIconCross" />
</template>
开启
