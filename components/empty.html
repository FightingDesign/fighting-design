Empty 空状态 ​

在展示空状态时的占位提示

  • 源代码
  • 文档编辑

基本使用 ​

content 属性可以配置展示的内容

在这里，后续提供更多服务
html
<f-empty content="在这里，后续提供更多服务" />
开启

内容文字大小 ​

content-size 属性可以配置 empty 内容的字体大小

暂无数据
html
<f-empty :content-size="25" :icon-size="50" />
开启

内容文字颜色 ​

content-color 属性可以配置 empty 内容文字的颜色

暂无数据
html
<f-empty content-color="orange" />
开启

自定义图片 ​

image-src 属性可以配置 empty 展示数据状态的图片

image-size 属性可以配置自定义的图片大小

暂无数据
html
<f-empty
  image-src="https://raw.githubusercontent.com/Tyh2001/images/master/fighting-design/1.jpg"
  image-size="200px"
/>
开启

自定义背景色 ​

background 属性可以配置 empty 的背景颜色

暂无数据
html
<f-empty background="#489EE2" image-size="300px" />
开启

自定义页脚 ​

footer 插槽可自定义页脚

暂无数据
我的数据哪去了？
html
<f-empty>
  <template #footer>我的数据哪去了？</template>
</f-empty>
开启

Attributes ​

参数说明类型可选值默认值
content展示的内容string——暂无数据
content-size内容文字大小string————
content-color内容文字颜色string————
image-src自定义图片地址string————
image-size自定义图片尺寸string————
icon-size自定义 icon 尺寸string——30
background自定义背景色string————

Slots ​

名称说明
default自定义插入内容
footer自定义页脚内容
